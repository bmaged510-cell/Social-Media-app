# 📱 Social Media Application

A full-featured social media platform built with **React.js** and **JavaScript ES6+** — featuring user profiles, dynamic feed, likes, comments, and real-time interactions.

🔗 **Live Demo:** [bmaged510-cell.github.io/Social-Media-app](https://bmaged510-cell.github.io/Social-Media-app/)

---

## ✨ Features

- 👤 User profiles with personal info
- 📰 Dynamic feed with real-time updates
- ❤️ Like & unlike posts
- 💬 Comments system
- 🔁 10+ interactive UI states
- 📱 Fully responsive design
- 🧩 Clean, modular component architecture

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| **React.js** | UI framework |
| **JavaScript ES6+** | Logic & async operations |
| **CSS3** | Styling |
| **React Hooks** | State management |
| **Async/Await** | API calls & data fetching |
| **REST APIs** | Backend communication |

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/bmaged510-cell/Social-Media-app.git

# Navigate to the project
cd Social-Media-app

# Install dependencies
npm install

# Run the development server
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
